## Introduction

This folder provides documentation, metadata, and data quality reports for the public education facilities dataset in Latin America and the Caribbean, available through the [SCL Data Lake](https://scldata.iadb.org/app/collections). It also includes exercises and scripts to facilitate analysis and exploration of these geospatial datasets. These resources are designed to help users understand the data, assess its quality, and apply it in analytical or policy-relevant contexts.

## Repository structure

- **dictionary.pdf**: provides a comprehensive data dictionary for the Educational Facilities dataset, including definitions of all variables, their data sources, accessibility status (public or private), and an overview of available variables.

- **/quality reports/**: contains quality control reports for each country, summarizing common data errors and cross-checking aggregated numbers with other sources. 

- **/exercises/**: R scripts with exercises related to the education facilities dataset. 

- **metadata.csv**: metadata for the Educational Facilities DATASET. 

## Data source

The files in this repository are based on the Education Facilities dataset from CIMA-IDB (2025). The dataset is available upon request via the [SCL Data Lake](https://scldata.iadb.org/app/folder/768E6E2A-DEFF-4BF9-B62D-E32518B15180). For access or further information, please contact the CIMA team: Elena Arias Ortiz (eariasortiz@iadb.org).
